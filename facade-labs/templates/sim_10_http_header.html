Gunakan cURL atau Burp Suite untuk mengirim request ke /api/v3/my-data dengan header yang dimodifikasi.

Request untuk user Anda (operator_one):

curl -H "X-User-ID: 1" http://127.0.0.1:5000/api/v3/my-data

Ubah nilai header menjadi 2 atau 3.
